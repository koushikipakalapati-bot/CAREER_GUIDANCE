# CAREER_GUIDANCE

A **Rule-Based Career Guidance System** built using **Python** and **Flask**.  
This web app suggests the **top 5 careers** based on user interests and provides **job description** and **demand/scope** for each career.


Features

- Keyword-based career recommendation (no ML, beginner-friendly)  
- Top 5 relevant careers displayed for any input  
- Job description for each career  
- Demand/scope information for each career  
- Clean and simple web interface  


 Technologies Used

- Python 3.x  
- Flask (Web Framework)  
- HTML / CSS (Frontend)  
- Rule-Based Logic for career recommendation
